# Fresk

Flutter frontend for the Fresk project !

## Project strucure
```
lib/
├── main.dart                # Entry point of the app
├── models/                  # Data models
├── widgets/                 # Custom widgets
├── screens/                 # App screens
│   └── home_screen.dart     # Screen with the graph
├── utils/                   # Utilities or helpers
└── assets/                  # Static assets like images and logos
```
